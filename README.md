# Research Portfolio — PhD Application, DEPOPLAND Project
## ETH Zurich, Chair of Planning of Landscape and Urban Systems (PLUS)

This repository links selected projects demonstrating experience in methodological areas central 
to the DEPOPLAND project: computational text analysis, spatial data methods, network analysis, 
and reproducible research pipelines.

---

## Projects

### 1. NLP pipeline for policy indicator extraction
**Repository:** [nlp-digital-agenda](https://github.com/mrosadio/nlp-digital-agenda)

Extracts and classifies structured policy indicators from heterogeneous, 
unstructured government documents across seven African countries using 
Python (pdfplumber, regex, fuzzy matching). Produces a comparative 
database from sources with inconsistent layouts and languages.

**Relevant to:** WP3 — deriving indicators from large text corpora

---

### 2. Spatial analysis of rainfall data
**Repository:** [risk-return-perception](https://github.com/mrosadio/risk-return-perception)

Descriptive spatial analysis linking TAMSAT rainfall data to Senegalese communes where data colletion was conducted for the Senegal Migration Panel to verify whether weather data was affecting responses on weather shocks perception. Analysis conducted using Python (xarray, GeoPandas).

**Relevant to:** WP1 — constructing spatially-explicit indicators from 
heterogeneous data sources

---

### 3. Migration network construction and diffusion analysis
**Repository:** [migration-networks](https://github.com/mrosadio/migration-networks)

Constructs social network edgelists from household survey data using R 
(igraph) and estimates spillover effects of information diffusion through 
social ties using PDS-LASSO and PPML. Integrated Stata/R pipeline, 
version-controlled on GitHub.

### 4. Climate finance diplomacy — data visualization
**Repository:** [map-climate-diplomacy](https://github.com/mrosadio/map-climate-diplomacy)

Country-level interactive visualization of climate finance partnership between 
African countries and the EU, Gulf States and China. Built with D3.js and a world GeoJSON dataset.

**Relevant to:** geospatial data communication and visualization

---

### 5. Impact evaluation using PDS-LASSO
**Repository:** [pdslasso-impact-evaluation](https://github.com/mrosadio/pdslasso-impact-evaluation)

Replication of a nonexperimental impact evaluation using the 
PDS-LASSO estimator for covariate selection in a youth employment 
program evaluation (Uganda).

**Relevant to:** quantitative and causal inference foundations
